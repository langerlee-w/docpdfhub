What kind of “Word” file do you get?

This converter focuses on text extraction. It reads the text layer of your PDF using PDF.js, then builds a clean .rtf document. RTF (Rich Text Format) opens in Microsoft Word, Apple Pages, Google Docs, and many other editors, so you can edit, copy, and reformat the content freely.

The main goal is to give you a quick way to get text back out of reports, whitepapers, academic articles, or contracts that were originally exported as PDF. The resulting file is much easier to work with than copying text page-by-page by hand.

Limitations (honest version)

  • Layout, columns, and precise spacing are not preserved — the output is linear text.
  • Tables may appear as lines of text separated by spaces or tabs.
  • Scanned PDFs (no text layer) can’t be converted unless OCR was run beforehand.
  • Images, logos, and diagrams are not included in the Word file.

For perfect, layout-accurate conversions you generally need a heavy desktop app or cloud service. This tool stays fully in your browser and keeps things lightweight and private.

Split PDF by Page Ranges & Extract Pages

Use comma-separated ranges like 1-3,5,8-10 to grab exactly what you need. Everything runs 100% locally in your browser.

Updated: 2025-11-08 · 6 min read

Why split by ranges?

Splitting by ranges lets you export only the sections you actually need. It’s perfect for sharing a chapter, sending a subset to teammates, or extracting attachments like forms and appendices.

Open the tool

Go to Split PDF. The tool works fully offline in your browser — your PDF never leaves your device.

Two modes

  • By page ranges — you control exactly which pages to export.
  • Each page into a file — creates one PDF per page, useful for quick batch extraction.

How to write ranges

Use commas to separate items. Both single pages and intervals are supported:

  • 5 → export only page 5
  • 1-3 → export pages 1, 2, 3
  • 1-3,8,10-12 → combine multiple segments

Ranges are 1-based (first page is 1). Reversed ranges like 7-4 are normalized to 4-7. Duplicates are removed automatically and merged when contiguous.

Step-by-step

  1. Drop your PDF into the drop zone or click Select PDF.
  2. Choose Split mode → By page ranges.
  3. Enter something like 1-3,5,8-10.
  4. (Optional) Tick Skip empty pages to avoid blank outputs.
  5. Click Add PDF to Split. You’ll get one file per continuous segment — e.g. pages-1-3.pdf, pages-5-5.pdf.

Naming & output

When your selection forms multiple segments, the tool exports multiple PDFs and auto-zips them. If there’s only one segment, it downloads directly as a single PDF for convenience.

Tips for reliability

  • Check the total page count first (e.g., if your doc has 12 pages, 13 will be ignored).
  • Use shorter segments for large scanned files to keep memory usage stable.
  • If a page renders blank, it’s often due to an empty or vector-only page — enable “Skip empty pages”.

Related tools

  • Merge PDF — Combine the exported segments back into a single file.
  • Compress PDF — Shrink the result before sending.
  • PDF to JPG — Convert the extracted pages to images.

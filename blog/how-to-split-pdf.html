How to Split a PDF by Ranges

Extract specific pages (e.g., 1–3, 5, 9–12) or split each page into its own file — fast & private.

Use Split PDF to divide large files into smaller parts, extract selections, or save one page per file — all locally in your browser.

Step 1 — Drop or Select a PDF

Click or drag a PDF into the box. DocPDFHub reads the file locally and detects total pages instantly.

Step 2 — Choose Split Mode

  • By ranges: Enter something like 1-3,5,9-12.
  • Each page: Make each page its own PDF.

Step 3 — Split Locally

Press Split PDF. Processing is done with browser-side code only; nothing leaves your device.

Step 4 — Download

You’ll get a ZIP of multiple files or a single output, depending on what you selected. Cleanly named, ready to share.

Tips

  • Combine with Merge PDF for quick rearrangement workflows.
  • Works offline after page load.
